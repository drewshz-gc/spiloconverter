= spilo2cnpg
:toc:
:toc-placement: preamble

Java-CLI: Analyse und Konvertierung von Zalando/Spilo-CRs nach CloudNativePG.

== Dokumentation

* *Vollstaendige CLI-Referenz:* `cloud-native-postgres-operator/documentation/tooling/spilo2cnpg.adoc`
* *Migrationsprojekt, Runbooks, Konzepte:* `cloud-native-postgres-operator/documentation/` (fuehrendes Repo)
* Governance: `cloud-native-postgres-operator/documentation/GOVERNANCE.adoc`

== Quick Start

[source,bash]
----
mvn -q package -DskipTests
java -jar target/spiloconverter-*.jar analyze --file path/to/postgres-cluster.yaml
java -jar target/spiloconverter-*.jar generate path/to/postgres-cluster.yaml -o cnpg/
----

== Build

[source,bash]
----
mvn package
mvn generate-resources   # cnpg-highlights-Redirect in src/main/doc (nicht mehr fuehrend)
----
